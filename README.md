# DarTwin Project Website

This is the website of the **DarTwin Project** - Foundations for the Systematic Evolution of Digital Twins.

The project is a collaboration between:
- **Johannes Kepler University Linz** (Institute for Business Informatics – Software Engineering)
- **University of Antwerp** (Cosys-Lab – Electronics-ICT)

Funded by the FWF WEAVE program.

## Website

This website is powered by [Jekyll](https://jekyllrb.com) and uses [Bootstrap](http://www.getbootstrap.com) and [Bootswatch](http://www.bootswatch.com). It is hosted on GitHub Pages at [dartwin-project.github.io](https://dartwin-project.github.io).

### Template

This website is based on the template from the [Allan Lab](https://www.allanlab.org) at Leiden University and LMU Munich. You can find their original template on [GitHub](https://github.com/allanlab/allanlab). We are grateful for their open-source contribution to the academic community.

## Local Development

To run the website locally:

```bash
docker compose up
```

Or using Jekyll directly:

```bash
bundle install
bundle exec jekyll serve
```

The website will be available at `http://localhost:4000`.
